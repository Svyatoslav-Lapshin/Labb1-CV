Bank Application

This project is a Console Application written in C# for a group project.
Features:

  • A UI library used for creating a good looking user interface inside the application.
  • Secure user storing using PBKDF2 hashing.
  • Dashboard and features for both a regular client and an admin.
  • Multiple different accounts such as Checking, Loan and Savings account.
  • Ability to make transactions between accounts.
  • Ability to take out loans.
  • 2-Factor Authentication.

MILTON

Team project application built during hackathon.
Features:

  • Language Agnostic: Supports Python, JavaScript, TypeScript, Java, C, C++, and C#.
  • Deterministic: Produces consistent output for the same input.
  • Rule-Based: Uses strict rules for component aggregation, avoiding AI hallucination.
  • Tree-sitter Powered: Uses robust incremental parsing.
  • Business Rule Extraction: Automatically detects validation logic and error conditions.
  • Smart Classification: Distinguishes between Domain, Orchestration, Interaction, and Infrastructure components.
